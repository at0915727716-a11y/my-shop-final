# 🏪 Absi stor - متجر إلكتروني متكامل

## 📋 عن المشروع
نظام متجر إلكتروني متكامل بقاعدة بيانات MongoDB ولوحة تحكم متقدمة.

### ✨ الميزات الرئيسية
- ✅ إدارة المنتجات (إضافة، تعديل، حذف) مع رفع الصور عبر Cloudinary
- ✅ سلة تسوق متطورة (localStorage)
- ✅ نظام طلبات متكامل مع واتساب
- ✅ كوبونات خصم وتقييمات المنتجات
- ✅ لوحة تحكم متجاوبة مع وضع ليلي (Dark Mode)
- ✅ دعم PWA (تثبيت كتطبيق)
- ✅ Google Analytics مدمج
- ✅ زر واتساب عائم للدعم الفوري
- ✅ تصدير الطلبات إلى CSV و PDF
- ✅ نسخ احتياطي تلقائي
- ✅ تحسين SEO (sitemap و meta tags)

## 🚀 طريقة التشغيل
1. `npm install`
2. `npm start` (أو `pm2 start ecosystem.config.js`)
3. افتح `http://localhost:3000/shop.html`
